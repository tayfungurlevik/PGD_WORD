PGD Word Add-in Kurulum

Sistem Gereksinimleri

  • Microsoft Word 2013 veya üzeri
  • .NET Framework 4.7.2 veya üzeri
  • Windows 7 SP1 veya üzeri

Kurulum Adımları

  1. Aşağıdaki "Yükle" butonuna tıklayın.
  2. Güvenlik uyarısı görürseniz "Çalıştır" veya "İzin Ver" seçeneğini tıklayın.
  3. Kurulum tamamlandığında Word'ü açın.
  4. "Ekle" sekmesinde PGD Add-in'i göreceksiniz.
Yükle

Sorun Giderme

  • Kurulum sırasında hata alırsanız, önce .NET Framework'ün en son sürümünü yüklediğinizden emin olun.
  • Word'de eklentiyi göremiyorsanız, Word'ü kapatıp yeniden açmayı deneyin.
  • Hala sorun yaşıyorsanız, sistem yöneticinize başvurun.
